# TLF RTF Generator

## Overview

Generating Tables, Listings, and Figures (TLFs) in RTF format often requires repetitive formatting and manual setup.  
This project provides an R function that automates the creation of TLF outputs directly into RTF files.

The tool is designed to simplify reporting workflows and reduce time spent on formatting tasks.

## Features

- Automatic title and footnote creation
- Dynamic file naming
- Page breaks based on grouping variables
- Multi-page output handling
- Automatic "(Cont’d)" page continuation labels
- Custom column headers
- Optional header underlining
- Column width and justification settings
- Flexible page layout options

## Benefits

- Save time on repetitive reporting tasks
- Standardize TLF outputs
- Improve efficiency in reporting workflows
- Focus on analysis instead of formatting
